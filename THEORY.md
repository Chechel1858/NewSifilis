# Теория: подавление периодического шума с помощью преобразования Фурье

## Введение

Цифровые изображения часто содержат различные виды шумов, возникающие при съемке, передаче или сканировании. Одним из наиболее сложных видов является **периодический шум** — регулярно повторяющийся узор в виде полос, сетки или муара.

В отличие от случайного шума, периодический шум имеет определенную частоту, поэтому наиболее эффективно устраняется в **частотной области**.

---

# Что такое периодический шум

Периодический шум представляет собой повторяющиеся структуры на изображении.

Причины возникновения:

- муар при сканировании;
- электромагнитные помехи;
- вибрации оборудования;
- особенности матриц камер;
- ошибки передачи данных.

---

# Пространственная и частотная области

Обычное изображение хранится в пространственной области, где каждый пиксель содержит значение яркости.

После преобразования Фурье изображение представляется как набор пространственных частот.

Низкие частоты отвечают за крупные объекты и общий фон.

Высокие частоты содержат мелкие детали и шум.

---

# Быстрое преобразование Фурье

Двумерное преобразование Фурье записывается как

F(u,v)=FFT(I(x,y))

В программе используется функция:

```python
f = np.fft.fft2(img)
```

---

# Центрирование спектра

После вычисления БПФ нулевая частота находится в углу изображения.

Для удобства анализа используют

```python
fshift = np.fft.fftshift(f)
```

Теперь низкие частоты находятся в центре спектра.

---

# Логарифмический спектр

Для отображения спектра используется

```python
magnitude = np.log1p(np.abs(fshift))
```

Логарифм делает слабые компоненты заметными.

---

# Как выглядит периодический шум

В спектре он проявляется в виде ярких симметричных точек относительно центра.

Именно эти пики необходимо удалить.

---

# Notch-фильтр

Создается маска:

```python
mask = np.ones_like(fshift)
```

Вокруг каждого обнаруженного пика вырезается небольшая область.

```python
mask[y-r:y+r, x-r:x+r] = 0
```

После применения маски шумовые частоты удаляются.

---

# Автоматический поиск пиков

Используется порог

```python
threshold = magnitude.mean() + 3 * magnitude.std()
```

Если яркость точки превышает этот порог, она считается шумовой.

---

# Обратное преобразование

После фильтрации выполняется

```python
img_back = np.fft.ifft2(np.fft.ifftshift(filtered_shift))
```

Получается очищенное изображение.

---

# Преимущества метода

- эффективно удаляет периодический шум;
- сохраняет детали изображения;
- подходит для автоматизации.

Недостатки:

- требует перехода в частотную область;
- неправильный выбор маски может удалить полезные детали.

---

# Заключение

Метод подавления периодического шума с использованием преобразования Фурье является одним из наиболее эффективных способов обработки изображений. Он широко применяется при обработке фотографий, медицинских снимков, спутниковых изображений и документов.
