# Подавление периодического шума на изображениях

## 📖 Описание

Данный проект демонстрирует способ подавления **периодического шума** на изображениях с использованием **быстрого преобразования Фурье (FFT)**.

Периодический шум проявляется в виде регулярных полос, сеток или муара и в частотной области выглядит как яркие пики, расположенные симметрично относительно центра спектра. Для его удаления применяется **режекторный (Notch) фильтр**, который вырезает соответствующие частоты.

---

# ⚙️ Как работает алгоритм

Алгоритм состоит из нескольких этапов.

### 1. Вычисление двумерного БПФ

Изображение переводится из пространственной области в частотную:

```text
F(u,v) = FFT(I(x,y))
```

В Python:

```python
f = np.fft.fft2(img)
```

---

### 2. Центрирование спектра

После вычисления БПФ нулевая частота находится в углу изображения.

Для удобства анализа спектр переносится в центр:

```python
fshift = np.fft.fftshift(f)
```

---

### 3. Построение логарифмического спектра

Чтобы сделать слабые частоты более заметными, вычисляется логарифмический спектр:

```python
magnitude = np.log1p(np.abs(fshift))
```

---

### 4. Поиск ярких пиков

Периодический шум проявляется как яркие точки на спектре.

Для их поиска используется порог:

```python
threshold = magnitude.mean() + 3 * magnitude.std()
```

Все точки, превышающие данный порог, считаются шумовыми.

---

### 5. Создание режекторного фильтра (Notch Filter)

Создаётся маска, заполненная единицами:

```python
mask = np.ones_like(fshift)
```

Вокруг каждого найденного пика формируется область, которая обнуляется:

```python
mask[y-r:y+r, x-r:x+r] = 0
```

Тем самым соответствующие частоты удаляются из спектра.

---

### 6. Применение фильтра

Маска умножается на спектр изображения:

```python
filtered_shift = fshift * mask
```

После этого шумовые частоты исчезают.

---

### 7. Обратное преобразование Фурье

Отфильтрованный спектр переводится обратно в изображение:

```python
img_back = np.fft.ifft2(...)
```

В результате получается изображение с подавленным периодическим шумом.

# 🖼 Что такое периодический шум?

Периодический шум — это регулярно повторяющийся узор на изображении.

Он может выглядеть как:

* горизонтальные полосы;
* вертикальные полосы;
* сетка;
* муар;
* повторяющиеся волны.

Например:

```text
#####################
#####################
#####################
```

или

```text
|||||||||||||||||||||
|||||||||||||||||||||
|||||||||||||||||||||
```

Такой шум гораздо эффективнее удалять в **частотной области**, чем обычными фильтрами размытия.

# 🚀 Подробное описание работы программы

## Шаг 1. Загрузка изображения

```python
img = cv2.imread("noise_image.png", cv2.IMREAD_GRAYSCALE)
```

Изображение загружается в оттенках серого.

---

## Шаг 2. Вычисление БПФ

```python
f = np.fft.fft2(img)
```

Изображение переводится в частотную область.

---

## Шаг 3. Центрирование спектра

```python
fshift = np.fft.fftshift(f)
```

После центрирования низкие частоты оказываются в центре изображения.

---

## Шаг 4. Построение спектра

```python
magnitude = np.log1p(np.abs(fshift))
```

Использование логарифма делает спектр более информативным.

Пример расположения пиков:

```text
      *
         *

    ●

 *         *
```

где

* **●** — центр спектра;
* ***** — шумовые частоты.

---

## Шаг 5. Поиск пиков

```python
threshold = magnitude.mean() + 3 * magnitude.std()
```

Вычисляется:

* среднее значение спектра;
* стандартное отклонение;
* автоматически определяется порог обнаружения.

---

## Шаг 6. Создание маски

```python
mask = np.ones_like(fshift)
```

После обнаружения пиков вокруг них вырезаются небольшие области.

Например:

```text
██████████████
███     ██████
███     ██████
███     ██████
██████████████
```

---

## Шаг 7. Фильтрация

```python
filtered_shift = fshift * mask
```

Все шумовые частоты становятся равны нулю.

---

## Шаг 8. Обратное БПФ

```python
img_back = np.fft.ifft2(...)
```

После обратного преобразования получается очищенное изображение.

# ⚙️ Настройка параметров

| Параметр                     | Описание                                                  |
| ---------------------------- | --------------------------------------------------------- |
| `center_radius = 30`         | Радиус центральной области, которую нельзя удалять.       |
| `threshold = mean + 3 * std` | Порог автоматического поиска шумовых пиков.               |
| `notch_size = 5`             | Размер вырезаемой области вокруг каждого найденного пика. |

При необходимости можно изменить параметры:

```python
threshold = magnitude.mean() + 2 * magnitude.std()
```

или

```python
threshold = magnitude.mean() + 4 * magnitude.std()
```

Также можно увеличить размер фильтра:

```python
notch_size = 10
```

если периодический шум выражен сильнее.

### 🖥 Особенности графического вывода

В зависимости от вашей операционной системы (наличия графической оболочки), поведение скрипта будет различаться:

#### А. Десктопный Linux (Ubuntu, Mint, Fedora и др. с GUI)
- При запуске откроется отдельное графическое окно.
- В нем будут отрисованы все три графика.
- ⏳ Скрипт будет работать (и "висеть" в терминале), пока вы не закроете это окно.

#### Б. Сервер без графики (Headless Linux / SSH / WSL)
- Оригинальный скрипт выдаст ошибку графического сервера (например, `_tkinter.TclError`), так как системе негде отрисовать окно.
- **Решение:** Замените последнюю строчку кода в `main.py` для сохранения результата в файл.

**Было:**
```python
plt.show()
```

**Стало:**
```python
plt.savefig("result_output.png", dpi=300)
```
После этого скрипт "молча" отработает и сохранит результат в файл `result_output.png` в текущую директорию.

## 🧠 Как это работает?
1. **Прямое БПФ**: Изображение переводится из пространственной в частотную область.
2. **Поиск пиков**: Яркие точки на спектре идентифицируются как периодический шум.
3. **Notch-фильтрация**: Найденные пики "зануляются" с помощью маски.
4. **Обратное БПФ**: Отфильтрованный спектр возвращается в пространственную область (чистое изображение).
